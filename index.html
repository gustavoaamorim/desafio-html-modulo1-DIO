História da Web e Primeiros Computadores

Evolução dos Computadores

A história dos computadores se inicou no ano de 1946 com o lançamento do Colossus e do ENIAC. Eles compõem a primeira geração de computadores.

Entre os anos de 1959 e 1965 surgem os computadores construídos com transistores, o que melhora a perfomance e aumenta a velocidade de cálculos realizados para microsegundos. Estes fazem parte da segunda geração de computadores.

A terceira geração engloba os anos de 1965 a 1970 com o uso de circuitos integrados para contrução dos computadores tornando-os ainda mais velozes na capacidade de processamento, desta vez com nanossegundos.

A partir do ano de 1975 inicia-se a quarta geração de computadores com compoentes cada vez mais miniaturizados através de circuitos integrados. Desde então os computadores passam a evoluir cada vez mais com a adoção de telas para inteface de usuário e componentes periféricos como teclados, mouse, autofalantes entre outros.

A quinta geração pode ser considerada a que nos encontramos atualmente com a criação de Inteligência Artificial (AI), metaverso e nuvem.

A História da Internet

A internet tem sua base na história da telecomunicação com a invenção do telégrafo e do telefone. Os primeiros passos e protótipos da internet eram restritos às comunicações militares. No final da década de 1950, com a Guerra Fria entre EUA e URSS, os EUA criaram a DARPA que foi fundamental para a história da internet ao desenvolver pesquisas e novas tecnologias, entre elas a rede de computadores.

No ano de 1962 J. C. R Licklider, então chefe da DARPA e considerado o pioneiro da internet, teorizou o conceito de uma rede galáctica de conexão onde seria possível acessar dados de qualquer lugar do mundo.

Já em 1969 aconteceu a primeira conexão entre dois computadores através da rede de comutação de pacotes, nascendo a ARPANET.

Entre os anos de 1970 e 1974 nasceram o NCP e o software básico de email. Durante este período a ARPANET começou a ser amplamente usada para conectar diversas universidades nos EUA, se afastando mais do uso militar.

Em 1974 foi desenvolvido um novo protocolo de comunimicação, o TCP/IP, que facilitava a comunicação entre as redes e em 1983 a ARPANET mudou para o novo protoco. Em 1985 a Internet já estava consolidada como a principal rede de comunicação de alcance global.

Em 1989, Tim Berners-Lee criou a World Wide Web unindo hipertexto com TCP/IP e assim surgiu o primeiro navegadr. O primeiro site criado pode ser conferido clicando aqui.

O que são Clients

Clients ou Clientes é a parte do ecossistema web que inicia a comunicação com o servidor. É também conhecido como o navegador.

Quando vamos realizar uma pesquisa no Google por exemplo, é enviada uma solicitação para o servidor com os dados que inserimos no campo de pesquisa, o servidor então analisa a solicitação e retorna com a informação pedida.

Os Clientes podem ser considerados toda a parte em que o usuário interage para buscar alguma informação ou usar algum serviço através da web.

O que são Servers

Os Servers ou Servidores são a parte por trás da web, algo como os bastidores. É onde ficam armazenadas todas as informações a serem disponibilizadas para os Clientes. O servidor recebe uma requisição do Cliente e retorna a informação desejada.

Uma analogia para entender melhor o conceito de Cliente-Servidor é imaginar um restaurante onde os Clientes são os próprios Clientes do restaurante que vão realizar um pedido, o garçom é a Internet que vai levar o pedido até a cozinha, que é o Servidor.

Linguagens de Programação

As linguagens de programação são um conjunto de instruções e regras usadas para gerar programas. Entre elas existem as linguagens orientadas para o Servidor (back-end) e as linguagens orientadas para o Cliente (front-end).

Cada linguagem apresenta uma sintaxe própria para construção do programa ou aplicação e podem ter similaridades entre elas. Alguns exemplos de linguagens de programação são Java, JavaScript, C++ e Phyton.

A linguagem JavaScript é uma linguagem de programação que pode ser usada tanto no lado do Servidor quando do lado do Cliente e é uma das mais usadas atualmente.

Já para criação de sites é usada a linguagem de marcação ou HTML que significa Hypertext Markup Language ou Linguagem de Marcação de Texto. Ela é utilizada junto com uma linguagem de estilização conhecida como CSS que significa Cascading Style Sheets ou Folha de Estilo em Cascata.

O que são Tags HTML

Como visto anteriormente, o HTML é uma linguagem de marcação para criação de páginas web. Esta marcação é feita atraves de elementos HTML que também são conhecidos como Tags. Através delas o navegador vai ler o cógido e exibir na tela com as devidas marcações. As tags possuem nomes específicos e são escritas ente < > como por exemplo a tag <p> que significa adicionar um parágrafo a página.

Exemplos de Tags

Entre as muitas tags existentes, as de título são usadas como muita frequência para definir títulos e subtítulos em páginas web e são definidas de h1 até h6. Veja abaixo um exemplo:

h1 Título 1

h2 Título 2

h3 Título 3

h4 Título 4

h5 Título 5
h6 Título 6

Outra tag usada é a de citação de bloco ou blockquote que gera o seguinte efeito:

Este é um exemplo de citação de bloco.

As tags <sub> e <sup> são usadas para criar efeitos de texto subescrito e sobrescrito como em C8H10N4O2 e E=mc2

A tag <font> define alguns parâmetros da fonte do conteúdo HTML como tamanho e cor por exemplo, porém é uma tag que já não é mais recomendada para uso.

Como último exemplo temos a tag <del> que representa uma parte do texto que foi excluída. É representada pelo navegador como uma linha "riscando" o texto deletado como no seguinte trecho este texto foi deletado.
